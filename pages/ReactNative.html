Documentação do Aplicativo React Native

Introdução

O aplicativo é usado como um painel de controle, dentro do aplicativo o usuário terá a possibilidade de comandar as ações dentro do jogo da Unreal Engine 5 por meio de requests HTTP

Login

Esta tela é a primeira a ser exibida ao usuário.
Nela, o usuário pode fazer login ou se cadastrar.

Perfil

Acessível atraves do primeiro ícone do rodapé.
Nessa tela é possível visualizar e editar as informações do concessionário como foto de perfil e telefone.

Seleção de família

Acessível atraves do segundo ícone do rodapé e também página inicial após o login.
Aqui o usuário pode selecionar a família de veículos que deseja No momento apenas a família Arocs está disponível.
Ao clicar em outras famílias aparecerá um aviso de que a família selecionada ainda não está disponível.
Ao clicar em uma família, o usuário é redirecionado para a tela de seleção de modelo.

Seleção de modelo

Acessível após selecionar a família na tela anterior.
Nessa tela, o usuário poderá selecionar entre os modelos disponíveis das famílias. No presente momento, todos levam para o mesmo lugar.
Ao clicar em um modelo, o usuário é redirecionado para a tela de ações.

Ações

Interno

Acessível atraves da barra de navegação superior. Ao entrar nessa tela, o usuário é teleportado para a parte interna do caminhão por meio de um request http.
No momento, as opções são apenas visuais, ao clicar em qualquer botão não ocorrerá nenhuma ação.

Externo

Acessível atraves da barra de navegação superior. Ao entrar nessa tela, o usuário é teleportado para a parte externa do caminhão por meio de um request http.
Todos os botões são funcionais, separados por categorias.

Estúdio

Acessível atraves da barra de navegação superior. Ao entrar nessa tela, o usuário é teleportado para o estúdio por meio de um request http.
No momento, as opções são apenas visuais, ao clicar em qualquer botão não ocorrerá nenhuma ação.
